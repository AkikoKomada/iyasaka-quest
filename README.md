# いやさかQUEST — コーちゃん完全自由版（Phase 2）

ドラクエ風タイルRPG。**マッチョ・リョウ・みいこの3人パーティ**で冒険。

## 公開URL（GitHub Pages）

**公開URL:** https://AkikoKomada.github.io/anirise-expression/

`gh-pages` ブランチにゲーム本体を push 済みです。

### 初回セットアップ（1回だけ）

1. GitHub リポジトリ → **Settings** → **Pages**
2. **Build and deployment** → Source: **Deploy from a branch**
3. Branch: **gh-pages** / **/ (root)** → Save
4. 2〜3分後、上記 URL で遊べます

※ ゲームを更新したあとは `main` に push 後、再度 `gh-pages` ブランチを更新してください（コーちゃんに依頼可）。

### ローカル確認

```bash
cd experiments/iya-quest-ko
npx serve . -p 5200
```

→ http://localhost:5200

## 操作

| キー | 動作 |
|------|------|
| ↑↓←→ / WASD | 4方向に歩く |
| Enter / Space / Z | 決定・会話・ドア |
| クリック | 決定 |

## Phase 2 — 3人パーティ

| キャラ | 役割 |
|--------|------|
| マッチョ | 操作キャラ（↑↓←→） |
| リョウ | 3歩遅れ（約1.5キャラ分） |
| みいこ | 6歩遅れ（約3キャラ分） |

- 会話中は3人とも停止
- タイトル：**いやさかQUEST**

## Phase 2 — 第2章「どうがを 届ける」

**北極星:** 人に価値を届けることで、仲間と未来が広がる  
**ゴール:** 10/31 横浜「異矢世界講演会」大盛況（第3章以降）

### 第2章の流れ

1. 村人と話す → アキコの家 → **いやさかのどうが** GET
2. **第2章イントロ**（ナレーション）
3. **コーちゃん**に話す → 告知の丘へ
4. **ウリボウ**に話す → ついてくる（パーティ追尾）
5. どうがむら **→ 東**（右端）→ **告知の丘**
6. 中央の **告知看板** に向かって Enter → どうがを貼る
7. **旅の青年**と話す → 応援者1人目
8. むらに戻り **むらびとA** → 2人目、**むらびとB** → 3人目
9. **コーちゃん** → 第2章クリア！

ステータスバー右上に **応援 N人** が表示されます。

## 世界観（自由設定）

- **いやせかい** … 動画の魔法が動く不思議な世界
- **どうがむら** … 冒険の拠点。川・橋・森・家々
- **NPC** … むらびとA/B、コーちゃん、ウリボウ、アキコ、旅の青年
- **クエスト（ゆるい）** … 村人と話す → アキコの家 → いやさかのどうが GET

## 技術

- 16×16 タイルマップ（40×24 野外 + 12×10 室内）
- 衝突判定・ドア遷移・向きを向けて会話
- キャラ：参考ドット絵PNG → 48px表示（移動グリッド16px・足元基準・輪郭）
- フラグ連動の会話分岐
